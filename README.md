# NetEase_Music

下载网易云音乐歌曲，依赖selenium以及requests

整个程序的核心是一个URL,感谢github让我找到了这个很重要的URL

>song_url= 'http://music.163.com/song/media/outer/url?id=***.mp3'
这个是下载的链接url,只要获取到歌曲的id就可以下载歌曲，虽然我也不知道这个url怎么会被找 到的      ^-^
